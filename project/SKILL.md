If creating visual artifacts (slides, mocks, throwaway prototypes, etc), copy assets out and create static HTML files for the user to view. Always import `colors_and_type.css` and apply the `kc` class on the root element to inherit the foundations.

If working on production code, you can copy assets and read the rules here to become an expert in designing with this brand.

Key things to remember:
- Kaia Cacao is a pre-launch ceremonial-cacao wellness brand. Tagline: "Shift Your Mood".
- The voice is warm, sensory, first-person plural ("we"). No emoji, no exclamation points, no health claims.
- Palette is six warm-leaning colors centered on deep teal, dusty rose, and warm cream. No bluish-purple gradients.
- Type pairing is Cormorant Garamond (display serif) + Red Hat Display (UI sans).
- Animation is gentle slow-outs only (260–520ms). No bounces or springs.
- Photography is golden-hour, warm, slightly grainy. People-first, not product-on-white.

If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.
